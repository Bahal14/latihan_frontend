# latihan_frontend

aplikasi untuk latihan murid calerate dan juga sebagai contoh dari project tugas yang akan di buat.

## fitur-fitur
- authentikasi dan middleware
- crud
- react router

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

## persyaratan
node 18+ (rekomendasi)
npm, pnpm, yarn 
## Cara install

Pastikan terlebih dahulu direktori kalian sudah masuk di masing masing folder latihan_frontend_api dan latihan_backend_api

Sebelum menjalankan project untuk install npm terlebih dahulu pada masing-masing folder itu

### `npm install`

Dilanjut dengan menjalankan project baik latihan_frontend_api maupun latihan_backend_api dengan perintah berikut.

### `npm start`
